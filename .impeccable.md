## Design Context

### Users
面向希望把 Claude、Cursor、Codex、Trae、VS Code 等各类 AI CLI 或开发工具快速接入 New API 中转站的开发者与折腾用户。使用场景通常是寻找可直接复制的配置方式、命令示例和工具差异说明，而不是阅读冗长的品牌介绍。

### Brand Personality
极简、专业、直接。页面应传达“这是一个能快速解决配置问题的开发者教程页”的感觉，减少营销话术，强调效率、可信度和可扩展性。

### Aesthetic Direction
单页 landing page，偏开发者工具站风格。整体保持清爽、克制、信息结构清晰。允许适度动效，但以轻量、顺滑、服务信息层级为主。页面需要天然适合后续继续加入更多工具，例如 Trae、Trae CN、Trae Solo、Trae Solo CN、VS Code 等。

### Design Principles
1. 单页讲清楚：所有核心信息集中在一个页面完成介绍与导航。
2. 教程优先：命令、配置方式、工具卡片和接入路径比装饰更重要。
3. 动效克制：只使用轻动效增强层级、反馈和滚动节奏，不做喧宾夺主的表演型动画。
4. 易于扩展：工具列表、分类结构和卡片组件必须方便继续增加更多工具。
5. 开发者友好：文案直接、信息密度适中、可快速扫描和复制。
